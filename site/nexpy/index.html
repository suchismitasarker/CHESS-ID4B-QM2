Nexpy¶

The code provides various tools for stacking the raw data, finding Bragg peaks, solving orientation matrix, and finally providing the HKL. The results are visible to NeXpy GUI.

Recommendation to take a look at data visulization package

Nexpy visulization Documentation

Steps to use the NexPy at beamline¶

Step 1 : Open the Nexpy

  • Terminal > ssh -Y lnx1034-f1

  • Terminal > nexpy &

  • Terminal > ssh lnx306
