PyMca¶

PyMca is developed by European Synchrotron Radiation Facility (ESRF). You can visulize and analyze the energy-dispersive X-ray fluorescence data. It access SPEC shared memory to monitor data acquisation. The user can see their plots in pyMCA.

For more information visit PyMca website
